Publication boundary

Structured facts, not private game data.

The public projection excludes ROMs, saves, screenshots, video, audio, coordinates, collision maps, prompts, raw model output, chat, identities, secrets, and local paths. Static files support GET and HEAD only; there is no server-side query or dynamic chat endpoint.
