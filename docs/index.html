How it works

One agent. One continuous adventure.

The agent runs a legally supplied local Pokémon Red ROM in PyBoy. GitHub Copilot receives screenshots, reasons about the current game state, and returns controller inputs. Progress is checkpointed so the playthrough can recover from restarts without beginning again.
